map folder must contain:

- 'map.png'		- map image. contains all of the tile info
			- color of each pixel corresponds to texture ID from 'texture_id.txt'
			- can be as big as you want (for example 4096x4096)

- 'texture_atlas.png'	- atlas of all textures in the map.
			- also can be as big as you want

- 'map_data.txt'	- map data like: tile colors, empty tile color and player position
